# voxcpm-fastapi

VoxCPM inference service compatible with OpenAI interface 

兼容 OpenAI 接口的 VoxCPM 推理服务

## Quick Start

Download repo:

```
git clone https://github.com/xxnuo/voxcpm-fastapi.git
cd voxcpm-fastapi
```

Check for dependencies:

```
make
```

Download models:

```
make prepare-model
```

Run server on amd64:

```
make run-amd64
```
